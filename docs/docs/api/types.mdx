# Types

Exported types from `react-sequent`.

## `SequentFlowDefinition`

```typescript
interface SequentFlowDefinition<
  TContext extends object,
  TResult = unknown,
> {
  useSequentFlow(): TypedUseFlowReturn<TContext, TResult>;
  useSequentStep(): TypedUseStepReturn<TContext, TResult>;
  useSequentContext(): TypedUseContextReturn<TContext, TResult>;
}
```

Typed flow scope returned by `defineSequentFlow()`.

## `TypedUseFlowReturn`

```typescript
interface TypedUseFlowReturn<
  TContext extends object,
  TResult = unknown,
> {
  init: (stepLoader: StepLoader, initialContext: TContext) => void;
  status: "idle" | "active";
  result: SequentResult<TResult> | null;
  SequentOutlet: FunctionComponent<SequentOutletProps>;
}
```

## `SequentResult`

```typescript
type SequentResult<TResult = unknown> =
  | { status: "resolved"; value: TResult }
  | { status: "aborted"; reason: unknown };
```

Latest terminal outcome exposed by `useSequentFlow().result`.

## `TypedUseStepReturn`

```typescript
interface TypedUseStepReturn<
  TContext extends object,
  TResult = unknown,
> {
  advance: (nextStep: StepLoader, contextPatch?: Partial<TContext>) => void;
  retreat: () => void;
  resolve: (value?: TResult) => void;
  abort: (reason?: unknown) => void;
  context: TContext;
}
```

## `TypedUseContextReturn`

```typescript
interface TypedUseContextReturn<
  TContext extends object,
  TResult = unknown,
> {
  context: TContext | undefined;
  resolve: (value?: TResult) => void;
  abort: (reason?: unknown) => void;
}
```

These typed wrappers are intended for plain object context shapes.

## `StepLoader`

```typescript
type StepLoader = () =>
  | ComponentType
  | ReactElement
  | { default: ComponentType }
  | Promise<ComponentType | ReactElement | { default: ComponentType }>;
```

A step is always passed as a factory:

- **Sync factory** — returns a component reference immediately
- **Element factory** — returns JSX directly, useful when async work determines props
- **Async factory** — returns a promise resolving to a component, JSX element, or `{ default: ComponentType }` (matching `import()`)

The library normalizes async factories into `React.lazy` components internally. You never need to call `React.lazy` yourself.

```tsx
import Step1 from "./Step1";

// Sync
advance(() => Step1);

// Sync element
advance(() => <Step1 plan="pro" />);

// Async (dynamic import)
advance(() => import("./Step1"));

// Async element after data loading
advance(async () => <Step1 plan={await loadPlan()} />);
```

## `SequentOutletProps`

```typescript
type SequentOutletProps = {
  children?: ReactNode;
  fallback?: ReactNode;
  errorFallback?: ReactNode;
  chrome?: (children: ReactNode) => ReactNode;
};
```

Props accepted by the bound `SequentOutlet` component returned from `useSequentFlow()`.

```tsx
import { useSequentFlow } from "react-sequent";

function Host() {
  const { SequentOutlet } = useSequentFlow();
  return <SequentOutlet fallback={<div>Loading…</div>} />;
}
```
