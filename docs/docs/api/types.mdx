# Types

Exported types from `react-sequent`.

## `StepLoader`

```typescript
type StepLoader = () =>
  | ComponentType
  | { default: ComponentType }
  | Promise<ComponentType | { default: ComponentType }>;
```

A step is always passed as a factory:

- **Sync factory** — returns a component reference immediately
- **Async factory** — returns a promise resolving to a component or `{ default: ComponentType }` (matching `import()`)

The library normalizes async factories into `React.lazy` components internally. You never need to call `React.lazy` yourself.

```tsx
import Step1 from "./Step1";

// Sync
advance(() => Step1);

// Async (dynamic import)
advance(() => import("./Step1"));
```

## `SequentOutletProps`

```typescript
type SequentOutletProps = {
  children?: ReactNode;
  fallback?: ReactNode;
  errorFallback?: ReactNode;
  chrome?: (children: ReactNode) => ReactNode;
};
```

Props accepted by the bound `SequentOutlet` component returned from `useSequentFlow()`.

```tsx
import { useSequentFlow } from "react-sequent";

function Host() {
  const { SequentOutlet } = useSequentFlow();
  return <SequentOutlet fallback={<div>Loading…</div>} />;
}
```
